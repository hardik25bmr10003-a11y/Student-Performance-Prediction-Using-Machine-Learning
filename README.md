# Student-Performance-Prediction-Using-Machine-Learning

## 📌 Overview
This project predicts student performance using machine learning algorithms based on various academic and personal factors such as study time, attendance, and previous scores.

## 🚀 Features
- Predict student scores
- Multiple machine learning models implemented
- Data visualization for insights
- Simple and easy-to-use structure

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📂 Project Structure
```bash
student-performance/
│── data/
│   └── dataset.csv
│── notebook/
│   └── model.ipynb
│── src/
│   └── train.py
│── requirements.txt
│── README.md
```

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/student-performance.git
```

### 2. Navigate into the project directory
```bash
cd student-performance
```

### 3. Install required dependencies
```bash
pip install -r requirements.txt
```

## ▶️ Usage

### Run the training script
```bash
python src/train.py
```

### Or open the Jupyter Notebook
```bash
jupyter notebook
```

## 📈 Models Used
- Linear Regression
